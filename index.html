My Methodology

Think

We research at what's already out there and how it can be improved. We explore areas that can be improved to create a framework to help guide your project.

Analyse

Analysis is a very powerful tool. We review the various ideas which are further streamlined into formats that suits your requirements, scope and target audience.

Design

"A good visual design of a website can determine a company success." Ideas filtered are used in creating visual appealing mockups which are converted into web pages.

Implement

In this stage, the web pages are built according the mockups design. This involues using the specified frameworks / tools required for your project.

Test

Testing can help you avoid costly mistake and get the most out of your web development efforts. It helps to ensure that the project meets all specified requirements / Scope.

Deploy

This is the final stage which involues deploying the files on a server inorder to make it accessible via the internet for your customers.
